### **II. THE WELDING & TESTING STATION (LANGUAGE: SQL)**

*Use these to update existing access or force a security audit.*

**[SQL] LINK ACCESS KEY TO EXISTING ACTOR**
UPDATE public.actor_db SET assigned_key = 'PASTE_NEW_KEY_HERE', status = 'Active' WHERE name = 'ACTOR_NAME_HERE';

**[SQL] TEST KEY EXPIRY (THE 48-HOUR TIME MACHINE)**
UPDATE public.site_keys SET created_at = NOW() - INTERVAL '48 hours' WHERE key = 'PASTE_KEY_HERE';
