## 5. CODING RULES (THE "LAW")

1. **Language:** JavaScript (`.js`) only. No TypeScript.
2. **Imports:** Use absolute (`@/components/...`) or clean relative (`../../`).
3. **Theme Logic:**
* **ALWAYS** import `useTheme` from `../../components/ui/ThemeContext`.
* **NEVER** hardcode colors. Use `style={{ color: currentColor }}`.


4. **Styles:**
* Use **Tailwind** for layout.
* Use **Inline Styles** for dynamic theme colors.
* Use **Globals.css** for complex animations (Shimmer, Scanlines).
